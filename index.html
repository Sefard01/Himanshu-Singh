    Himanshu Singh — Developer, Writer, Digital Artist

    Main Himanshu Singh hoon. Simple banda, lekin dimaag thoda overthinker type hai.

    Main developer bhi hoon aur writer bhi. Matlab din me code likhta hoon aur raat me thoughts. Mujhe cheezein banana pasand hai — websites, ideas, theories… jo bhi dimaag me aaye usko reality me convert karna mera kaam hai.

    Meri website basically mera digital notebook hai. Yahin main apne projects, books, experiments aur random deep thoughts daalta hoon.

    Latest Writing

      About Himanshu Singh

      Himanshu Singh

      Developer × Author

      Main products logic se banata hoon, aur logon tak words se pahuchta hoon.

      Main full-stack developer hoon — JavaScript, web apps, logic, debugging… ye sab meri daily life ka part hai. Agar koi cheez toot jaaye toh main use fix karne me maza leta hoon. Aur agar koi system inefficient ho toh usko optimize karna mera hobby hai.

      Mera goal simple hai — complex cheezon ko simple banana. Chahe wo code ho ya human emotions.

      Aur haan, main job ke peeche bhaagne wala banda nahi hoon. Mujhe apna system banana hai. Apni identity create karni hai. Long game khelna hai.

      Bas itna samajh lo — main abhi starting phase me hoon, lekin direction clear hai. Build. Write. Think. Repeat. 🚀

      Agar tum mere dost ho, toh ready rehna — kabhi startup idea discuss hoga, kabhi life philosophy, aur kabhi 2 baje raat ko “ek crazy idea aaya hai” wala message 😂
